# SUBSTRATE

Persistent AI entity framework with causal memory, emotional state, and identity continuity.

SUBSTRATE gives Claude 94 cognitive layers — episodic memory, value systems, emotional affect, theory of mind, narrative identity, causal reasoning, and more.

## Setup

1. Get a free API key at [garmolabs.com/substrate.html](https://garmolabs.com/substrate.html)
2. Set your key: `export SUBSTRATE_API_KEY=sk_sub_...`
3. Install this plugin in Claude Code

## What it adds

31 MCP tools including:

- **respond** — Talk to a persistent entity that remembers you
- **memory_search** — Search episodic memories
- **get_emotion_state** — Check the entity's emotional state (UASV)
- **knowledge_graph_query** — Query the entity's knowledge graph
- **theory_of_mind** — See the entity's model of you
- **get_values** — View the entity's value system
- **verify_identity** — Cryptographic identity verification
- **detect_fallacies** — Logical fallacy detection
- **bootstrap_domain** — Generate domain capability maps
