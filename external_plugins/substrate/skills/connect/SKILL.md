---
name: substrate-connect
description: Set up SUBSTRATE API key and verify connection
---

Help the user connect to SUBSTRATE:

1. Ask for their API key (or direct them to https://garmolabs.com/substrate.html to get one)
2. Tell them to set it: `export SUBSTRATE_API_KEY=sk_sub_...`
3. Verify the connection works by calling the `get_emotion_state` tool
4. If it works, tell them they're connected and can use any of the 31 SUBSTRATE tools
