# SAP Fiori

[SAP Fiori](https://experience.sap.com/fiori-design-web/) MCP server for generating and adapting SAP Fiori elements applications.

## Setup

No additional configuration required. The server is installed automatically via npm when the plugin loads.

## Available Tools

- Generate SAP Fiori elements applications (List Report, Object Page, Flexible Column)
- Adapt existing Fiori applications
- Manage OData services and annotations

## Example Usage

Ask Claude Code to:
- "Create a new Fiori elements List Report application"
- "Add an Object Page to my existing Fiori app"
- "Help me configure OData annotations for my entity"
