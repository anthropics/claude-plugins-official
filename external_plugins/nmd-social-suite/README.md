# NMD Social Suite

One-command social media content engine built by NMD ZAZA.

## What it does

Type `/social` and Claude creates + posts content across every platform simultaneously:

- Instagram captions
- Facebook posts
- X/Twitter threads
- LinkedIn articles
- Full blog posts (SEO)
- Email newsletter editions
- Credit/finance forum posts
- AI image prompts
- AI video scripts

Auto-posts via your browser. No API keys. No third-party apps.

## Setup

Run once:
```
/social-setup
```

Then every session:
```
/social                          ← full platform blast
/social blog about [topic]       ← long-form blog only
/social newsletter [topic]       ← newsletter edition only
/social repurpose                ← paste content, adapt for all platforms
/social forums                   ← credit/finance forum posts only
/social video [url]              ← AI video campaign from URL
```
