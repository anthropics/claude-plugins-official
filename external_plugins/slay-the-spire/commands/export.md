# Export Run

Invoke @spire:spire to export your current run as a formatted markdown tracker.

## Usage

```
/slay-the-spire:export
```

This will:
- Generate a markdown run tracker with your full run history
- Include deck composition, relics obtained, floors cleared
- Save to a file in your chosen directory
- Format for easy sharing on Discord, Reddit, or guides

**Optional:** Specify output directory:
```
/slay-the-spire:export ~/Desktop/
```
