# mumcp — WordPress MCP Plugin for Claude Code

Connect Claude Code to any WordPress site with 239 MCP tools covering pages, Elementor, WooCommerce, media, SEO, and more.

## Features

- **239 MCP tools** across pages, posts, Elementor, WooCommerce, media, SEO, settings, and more
- **Role-scoped API keys** — admin, designer, editor, author, or custom tool subsets
- **Elementor visual builder** — read and write Elementor page layouts via natural language
- **WooCommerce** — manage products, orders, customers, and inventory
- **Self-updating** — the WordPress plugin updates itself over the air

## Installation

### WordPress Plugin

Install the WordPress plugin on your site from the [WordPress.org directory](https://wordpress.org/plugins/mumcp) or via:

```bash
wp plugin install mumcp --activate
```

### Claude Code Plugin

```bash
/plugin install mumcp@claude-plugins-official
```

Then connect to your site using the `/mumcp` skill.
