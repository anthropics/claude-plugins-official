# plugin-browser

A local web UI for browsing the official Claude Code plugin marketplace.

Browse all plugins, filter by type, sort by popularity, and copy install commands with one click. Runs entirely offline using your local `~/.claude/plugins/` directory — no network required.

## Install

```
/plugin marketplace add devycelabs/claude-plugin-browser
/plugin install plugin-browser
```

Then use `/plugin-browser` to open the UI.
