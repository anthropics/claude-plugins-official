# FlowHunt Skill

Automation discovery audit for Claude Code. Walks through a 5-question workflow intake and audits connected apps (Gmail, Calendar, Slack, task trackers) to identify automation opportunities.

## Installation

```bash
npx skills add heyneuron/flowhunt-skill
```

## What It Does

1. Runs a 5-question intake to understand your current tool stack and pain points
2. Audits your connected integrations (Gmail, Google Calendar, Slack, Asana, Linear, Jira, Trello) for automation signals
3. Returns a prioritized automation roadmap with quick wins, medium projects, and strategic automations
