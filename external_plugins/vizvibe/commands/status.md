# Viz Vibe Status

Show the current status of the project trajectory.

## Steps:

1. Read `vizvibe.mmd` file

2. Extract and display:

   - **Last Active Node**: What was worked on most recently
   - **Open Tasks**: All nodes with `[opened]` state
   - **Recently Closed**: Last 3 closed nodes
   - **Total Nodes**: Count of all nodes

3. Show a brief summary of:

   - Project goals (ultimate and current)
   - Current focus area
   - Any blockers

4. Format as a clear, scannable summary
