[Plugin] Clypt — Link Intelligence

## PR Description
Clypt is an AI-native link intelligence platform. This plugin enables Claude to:
- Create short links with custom slugs, UTM parameters, tags, and folders
- Launch full campaign link sets (email + social + ads) in one command
- Pull real-time analytics — clicks, geo, device, referrer
- Generate and customise QR codes (PNG or SVG)
- Bulk-create up to 25 links at once
- Update destinations without breaking analytics or QR codes

**npm package:** clypt-mcp@1.1.1 (open source, MIT)
**MCP Registry:** io.github.Bonsai-Tech-LLC/clypt
**Docs:** https://clypt.io/developers
