# community-ops

A Claude Code skill for building community ops platforms — bidirectional systems that aggregate data from multiple community tools and push actions back into them.

## What It Teaches

Most community tools are read-only monitors. This skill shows you how to build a **bidirectional hub** where community managers see a full 360 view of every customer interaction and can take action without switching tabs.

## Install

```
/plugin install community-ops@claude-plugins-official
```

## What You'll Build

- Single-surface aggregation of Discord, Slack, ticketing, CRM, and billing data
- Auto-triage with LLM classification and human override
- Bidirectional actions: reply, create tickets, flag threads, post digests — all from one surface
- Customer 360 view: tickets, account status, billing tier, prior history
