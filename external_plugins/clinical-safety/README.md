# Clinical Safety Plugin

**By [Digital Clinical Safety](https://www.digitalclinicalsafety.com)**

AI-assisted clinical safety pre-review for health IT products against
DCB0129, DCB0160, ISO 27001, IEC 62304, and ISO 14971.

Reads your codebase and Notion/Confluence product documentation to produce
a prioritised list of questions and observations for your Clinical Safety Officer.

## Install
```bash
claude plugin add github:YOUR-USERNAME/clinical-safety-plugin
```

## What It Does

- Reviews code changes for clinical safety hazards
- Pulls product specs from Notion or Confluence via MCP
- Produces 🔴 Priority / 🟡 Process / 🟢 Observation reports for your CSO
- Includes a CSO-perspective subagent for meeting prep

## Standards Covered

DCB0129 · DCB0160 · ISO 27001 · IEC 62304 · ISO 14971
