# SAP UI5

[SAPUI5/OpenUI5](https://ui5.sap.com/) MCP server for creating and validating UI5 applications.

## Setup

No additional configuration required. The server is installed automatically via npm when the plugin loads.

## Available Tools

- Create and validate UI5 applications
- Access SAPUI5 API documentation
- Run UI5 linter for code quality checks
- Get UI5 development guidelines and best practices

## Example Usage

Ask Claude Code to:
- "Create a new UI5 application with a table view"
- "Run the UI5 linter on my project"
- "Show me the API docs for sap.m.Table"
