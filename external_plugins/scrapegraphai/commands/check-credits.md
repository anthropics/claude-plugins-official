# Check Credits Command

Check your current ScrapeGraphAI credit balance and total usage.

## Usage

```
/check-credits
```

## Examples

```
/check-credits
```

## Response

Returns:
- **remaining_credits**: Number of credits remaining
- **total_credits_used**: Total credits consumed

## Notes

- Requires `SCRAPEGRAPHAI_API_KEY` environment variable
- No credits consumed for this operation
- Useful before starting large scraping operations
