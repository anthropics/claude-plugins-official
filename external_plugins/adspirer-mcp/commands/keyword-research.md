# Keyword Research

Perform keyword research for Google Ads campaigns:

1. Accept the business description or seed keywords from the arguments
2. Call the `research_keywords` tool with:
   - Business description or seed keywords
   - Website URL if mentioned
   - Target location if specified
3. Present results in a formatted table showing:
   - Keyword suggestions grouped by intent (high/medium/low)
   - Monthly search volume estimates
   - CPC ranges and competition level
   - Budget recommendations (conservative/moderate/aggressive)
4. Recommend which keywords to target based on intent and budget
5. Suggest negative keywords to exclude
