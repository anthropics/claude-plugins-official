# okr-creator

AI-powered OKR Creator — analyzes any project with six-dimension diagnosis, generates harness-driven OKR skill with PUA rhetoric, and deploys daily GitHub Action review with issue-based AI chat.

A [PUA](https://github.com/tanweai/pua)-like project — PUA makes AI never give up, OKR tells AI where to push.

## Features

- **Six-dimension diagnosis** — Vision, Quality, Debt, Architecture, Docs, Automation
- **PUA-style intent elicitation** — Forces users to clarify priorities and deadlines
- **Harness-driven KRs** — Every Key Result has a verifiable acceptance method
- **Auto-deploy GitHub Action** — Daily OKR review + issue-based `@claude`/`@codex` chat
- **Auto Mode** — Non-interactive execution for CI/CD pipelines

## Usage

```
/okr
```
