# KreatSaaS

SaaS Builder Guide - Build production-ready SaaS from design to deployment.

## Features

- **11 Skill Guides**: beginner, setup, admin, SEO, deployment, modification, FAQ, and more
- **5 AI Agents**: architect, designer, developer, deployer, branding
- **5 Slash Commands**: /kreatsaas, /update-llm, /upgrade-models, /version, /request-feature
- **Multi-language**: Korean and English support
- **Beginner Friendly**: "Where do I configure this?" FAQ guide

## Installation

```bash
claude plugin add johunsang/kreatsaas
```

## Usage

```bash
/kreatsaas              # Start SaaS building workflow
/kreatsaas beginner     # Beginner guide
/kreatsaas setup        # Setup guide
```
