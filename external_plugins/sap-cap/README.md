# SAP CAP

[SAP Cloud Application Programming Model (CAP)](https://cap.cloud.sap/) MCP server for AI-assisted development of CAP applications.

## Setup

No additional configuration required. The server is installed automatically via npm when the plugin loads.

## Available Tools

- Search CDS model definitions
- Query CAP documentation
- Get AI-assisted development support for CAP applications

## Example Usage

Ask Claude Code to:
- "Show me the CDS entities in my project"
- "How do I define an association in CAP CDS?"
- "Help me create a new CAP service"
