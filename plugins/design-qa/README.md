# design-qa

Design QA and critique for frontend interfaces.

- Scores UI against Nielsen 10 heuristics (0-40 scale)
- Detects 31 AI-generated slop patterns
- Deterministic regex scanner (no LLM needed for 14/31 patterns)
- Component-level calibration
- Eval suite for regression testing

## Install

\`\`\`
npx skills add https://github.com/alex410000/design-qa
