# Cancel Ralph Loop

Execute the cancel script to stop the Ralph loop:

```!
"${CLAUDE_PLUGIN_ROOT}/scripts/cancel-ralph-loop.sh"
```

The Ralph loop has been cancelled. You can now exit normally or start a new loop.
