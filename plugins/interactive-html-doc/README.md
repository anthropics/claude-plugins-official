# Interactive HTML Document Plugin

Converts Markdown technical documents into polished, interactive single-file HTML pages with SVG diagrams, collapsible sections, animated step-throughs, and responsive navigation.

## Usage

```
"Convert this markdown to an interactive HTML document"
"Create an interactive HTML page from my notes"
"Turn this technical doc into an interactive webpage"
```

## Features

- Dark-themed, single-file HTML (no external dependencies)
- SVG vector diagrams replacing ASCII art
- Collapsible/expandable sections
- Interactive step-through animations
- Tab-based content switching
- Sticky sidebar table of contents with scroll tracking
- Mobile-responsive with hamburger menu
- Syntax-highlighted code blocks
