# liquid-lsp

Shopify Liquid language server for Claude Code, providing code intelligence for `.liquid` theme files.

## Supported Extensions
`.liquid`

## Installation

The [Shopify CLI](https://shopify.dev/docs/api/shopify-cli) must be installed and available in your `$PATH`.

### Via npm (recommended)
```bash
npm install -g @shopify/cli
```

### Via Homebrew (macOS)
```bash
brew tap shopify/shopify
brew install shopify-cli
```

Verify installation:
```bash
shopify theme language-server --help
```
