# scala-lsp

Scala language server for Claude Code, providing code intelligence, refactoring, and analysis.

## Supported Extensions
`.sbt`, `.sc`, `.scala`

## Installation

Install metals using either Homebrew or Coursier:
### Homebrew

```bash
brew install metals
```

### Coursier
```bash
cs install metals
```
