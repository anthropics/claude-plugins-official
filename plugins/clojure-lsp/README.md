# clojure-lsp

Clojure/ClojureScript language server for Claude Code, providing code intelligence, navigation, and refactoring.

## Supported Extensions
`.clj`, `.cljs`, `.cljc`, `.edn`

## Installation

Install clojure-lsp via Homebrew (macOS):

```bash
brew install clojure-lsp/brew/clojure-lsp-native
```

Or download a binary from the [GitHub releases](https://github.com/clojure-lsp/clojure-lsp/releases).
