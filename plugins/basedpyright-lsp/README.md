# basedpyright-lsp

Python language server (basedpyright) for Claude Code, providing stricter type checking and code intelligence. A community fork of Pyright with additional diagnostic rules and better error messages.

## Supported Extensions
`.py`, `.pyi`

## Installation

Install basedpyright with pip:

```bash
pip install basedpyright
```

Or with uv:

```bash
uv tool install basedpyright
```

Or with Homebrew:

```bash
brew install basedpyright
```
