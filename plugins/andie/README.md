# Andie

> Disciplined thinking skill for people who build with AI.

Andie routes every problem into the right thinking mode before AI acts. Four playbooks, named expert triads, HITL gates, and continuous OODA loops. Get it right, not just fast.

## Modes

| Mode | When to use |
|------|-------------|
| **Deep** | Understand, explain, teach something clearly |
| **Drama** | Debate a serious decision from multiple angles |
| **War** | Firefight — triage an urgent problem fast |
| **Kaizen** | Root-cause what keeps breaking |

## Install

```bash
npx andie-skill
```

Or copy `skills/andie/SKILL.md` to `~/.claude/skills/andie/SKILL.md`.

Then in Claude Code, type `andie` to start.
