# Beads Plugin

Slash commands for [beads](https://github.com/steveyegge/beads), a git-backed issue tracker with first-class dependency support.

## Commands

- `/bd-create` - Create a new issue
- `/bd-list` - List issues
- `/bd-show` - Show issue details
- `/bd-close` - Close an issue
- `/bd-graph` - Show dependency graph
- `/bd-ready` - Show ready issues (no blockers)

## Requirements

The `bd` binary must be installed and available in PATH.
