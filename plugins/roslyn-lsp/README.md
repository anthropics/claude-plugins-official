# roslyn-lsp

Microsoft's official Roslyn-based C# language server for Claude Code, providing rich code intelligence and diagnostics. This is the same language server that powers the C# extension for Visual Studio Code and C# Dev Kit.

## Supported Extensions
`.cs`

## Installation

### Via .NET tool (recommended)
```bash
dotnet tool install --global roslyn-language-server
```

## Requirements
- .NET SDK 10.0 or later
