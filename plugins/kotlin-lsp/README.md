# kotlin-lsp

> **Note:** This language server is currently in pre-alpha.

Kotlin language server for Claude Code, providing code intelligence, refactoring, and analysis.

## Supported Extensions
`.kt`, `.kts`

## Installation

### Homebrew

Install kotlin-lsp CLI using Homebrew:

```bash
brew install JetBrains/utils/kotlin-lsp
```

### VS Code

There is also a VSC extension available. See the [GitHub repository](https://github.com/Kotlin/kotlin-lsp) for installation instructions.

## Requirements
- JDK 17 or later
