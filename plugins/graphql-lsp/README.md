# graphql-lsp

GraphQL language server for Claude Code, providing schema and operation intelligence.

## Supported Extensions
`.graphql`, `.gql`, `.graphqls`

## Installation

Install the GraphQL language server globally via npm:

```bash
npm install -g graphql-language-service-cli
```
