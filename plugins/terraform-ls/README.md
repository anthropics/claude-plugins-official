# terraform-ls

Terraform language server for Claude Code, providing code intelligence and analysis.

## Supported Extensions

`.tf`, `.tfvars`

## Installation

### Homebrew (macOS / Linux)

```bash
brew install hashicorp/tap/terraform-ls
```

### Linux (apt/rpm)

Follow the [Official HashiCorp Packaging Guide](https://www.hashicorp.com/official-packaging-guide) to install from official repositories. The package name is `terraform-ls`.

### Other platforms

Download from [HashiCorp releases](https://releases.hashicorp.com/terraform-ls/) and add to your PATH.

Verify installation:

```bash
terraform-ls -v
```
