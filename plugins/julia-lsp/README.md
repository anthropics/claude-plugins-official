Julia language server for Claude Code, providing code intelligence, diagnostics, and navigation via LanguageServer.jl.

## Supported Extensions
`.jl`

## Installation

Install LanguageServer.jl in your Julia environment:

```julia
using Pkg; Pkg.add("LanguageServer")
```
