# lean4-lsp

Lean 4 language server for Claude Code, providing code intelligence for theorem proving.

## Supported Extensions
`.lean`

## Installation

Install Lean 4 via elan:

```bash
curl https://elan.lean-lang.org -sSf | sh
```
